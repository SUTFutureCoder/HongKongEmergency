# 香港生活与职场粤语速查表

> 面向在香港生活、办事和工作的中文使用者：先记能立刻开口的短句，再按场景补词。  
> 粤语使用香港常见口语字，读音采用粤拼（Jyutping）并标数字声调。  
> 标记：★★★ 高频必会；★★ 常用；★ 场景词。

---

## 0. 使用方法

这份速查表的目标是先让沟通继续，而不是一次学完所有语法：

- 听不清时先说 `唔好意思，可唔可以講慢啲？`
- 不确定时先确认 `即係……，係咪？`
- 需要对方帮忙时用 `唔該` 或 `麻煩你`
- 工作中把口语短句和书面消息分开：当面可以自然，邮件和文件仍用正式中文

### 粤拼速记

| 项目 | 说明 |
|---|---|
| 声调 | 粤拼数字 1 至 6 表示声调；同一个音节换声调可能完全变词。 |
| `aa / a` | `aa` 是长元音，如 `家 gaa1`；`a` 是短元音，如 `咁 gam2`。 |
| `j` | 按英语 `y` 的音读，如 `有 jau5`、`飲 jam2`。 |
| `z / c` | `z` 接近普通话 `j`，`c` 接近 `ts`，如 `早 zou2`、`錢 cin2`。 |
| 口语字 | `冇、唔、喺、佢、哋、嘅、咗、緊、啲` 在香港口语很常见。 |

### 两个容易混淆的感谢

| 粵語 | 粵拼 | 普通话 |
|---|---|---|
| 唔該 | m4 goi1 | 谢谢别人提供服务、帮忙；也可用于“请”。 |
| 多謝 | do1 ze6 | 谢谢别人送礼物、请客或比较郑重的感谢。 |
| 唔該晒 | m4 goi1 saai3 | 非常感谢（服务、帮忙）。 |
| 多謝你 | do1 ze6 nei5 | 谢谢你（礼物、心意或正式感谢）。 |

## 21. 最后提醒

- `唔該` 是香港日常最万能的礼貌词；开口前后都可以用。
- 听不懂时不要只说“啊”，直接说 `可唔可以講慢啲？`，对方通常会放慢。
- 口语中的 `咗` 表示已经发生，`緊` 表示正在进行，`過` 表示曾经发生过：`做咗`、`做緊`、`做過`。
- 职场中可以先用粤语短句建立沟通，再用普通话或英文确认复杂细节；重要决定仍应留书面记录。
